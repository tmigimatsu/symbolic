symbolic
========

.. _symbolic: https://github.com/tmigimatsu/symbolic
.. _Planning Domain Definition Language: https://planning.wiki/_citedpapers/pddl1998.pdf
.. _VAL: https://github.com/KCL-Planning/VAL

`symbolic`_ is a C++/Python library for parsing and manipulating `Planning
Domain Definition Language`_ (PDDL) symbols for AI planning. This library is
built upon `VAL`_, a C++ library for validating PDDL specifications.
